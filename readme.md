# 💡 IdeaShare Server

The backend powering **IdeaShare**, a full-stack platform where users can create, share, and discover meaningful ideas and experiences. The server provides secure authentication, RESTful APIs, payment processing, premium content management, and administrative controls.

## 🌐 API

**Live API:** https://idea-share-server.vercel.app/

---

# ✨ Features

## 🔐 Authentication

- Secure authentication with BetterAuth
- Session management
- Protected API routes
- Role-based authorization
- Server-side authentication validation

---

## 👤 User Management

- User registration and login
- Profile management
- User role management
- Admin role assignment
- User account removal

---

## 💡 Idea Management

- Create new ideas
- Read all published ideas
- Update existing ideas
- Delete ideas
- Featured ideas support
- Premium content management

---

## 💬 Community Features

- Like ideas
- Save ideas
- Comment system
- Report inappropriate content
- User activity tracking

---

## 💳 Payment System

- Stripe payment integration
- Secure checkout session creation
- Premium access verification
- Automatic premium content unlocking
- Payment validation

---

## 🛡️ Admin Features

- Manage all users
- Manage all ideas
- Review user reports
- Feature or unfeature ideas
- Moderate community content
- Maintain platform quality

---

## ⚙️ API Features

- RESTful API architecture
- Secure CRUD operations
- MongoDB database integration
- Server-side validation
- Error handling middleware
- Environment-based configuration
- JSON API responses
